# backend

## Project setup
```
npm install
```

### Start server and database
- in Linux or Mac
```
npm run start
```
- in windows, you need two terminals to run the following two commands respectively
```
npm run serve
```
```
npm run database
```

### Start dev server and database
- in Linux or Mac
```
npm run dev
```
- in windows
```
npm run devserve
```
```
npm run database
```

### Lints and fixes files
```
npm run lint
```

### Run unit tests
```
npm run test
```

### Init or reset database
```
npm run reset-db
```
